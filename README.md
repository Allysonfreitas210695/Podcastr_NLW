🚀 Getting started
Clone the project and access the folder

$ git clone https://github.com/guilhermecapitao/podcastr.git && cd podcastr
Follow the steps below

# Install the dependencies
$ yarn

# Run the JSON server
$ yarn server

# Start the project
$ yarn dev
